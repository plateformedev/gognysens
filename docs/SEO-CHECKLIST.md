# Checklist SEO COGNISENS

## 1. SEO Technique

### Structure HTML

- [ ] Balise title unique par page (< 60 caracteres)
- [ ] Meta description unique par page (< 155 caracteres)
- [ ] Un seul H1 par page
- [ ] Hierarchie des titres (H1 > H2 > H3)
- [ ] Balises alt sur toutes les images
- [ ] URLs propres et descriptives

### Performance

- [ ] Core Web Vitals au vert
- [ ] LCP < 2.5s
- [ ] FID < 100ms
- [ ] CLS < 0.1
- [ ] Score Lighthouse > 90

### Indexation

- [ ] Sitemap XML genere
- [ ] Sitemap soumis a Google Search Console
- [ ] Robots.txt configure
- [ ] Pas de pages en noindex involontaire
- [ ] Canonicals en place

## 2. Schema.org

### Global

- [ ] Schema Organization sur toutes les pages
- [ ] Donnees structurees valides (test Google)

### Par type de page

- [ ] LocalBusiness sur pages GEO
- [ ] Service sur pages expertise/AMO
- [ ] FAQPage sur pages avec FAQ
- [ ] BreadcrumbList sur toutes les pages internes

## 3. Contenu

### Pages principales

- [ ] Contenu unique et de qualite (min 500 mots)
- [ ] Mots-cles principaux dans H1, H2, premier paragraphe
- [ ] Variations de mots-cles
- [ ] Liens internes pertinents (3-5 par page)
- [ ] CTA clair

### FAQ

- [ ] 3-5 questions par page
- [ ] Reponses completes
- [ ] Schema FAQPage integre
- [ ] Questions en langage naturel

## 4. GEO SEO

### Pages locales

- [ ] Titre avec lieu (Paris, 92, 94, 78)
- [ ] Meta description avec lieu
- [ ] Contenu localise
- [ ] Schema LocalBusiness

### NAP (Name, Address, Phone)

- [ ] Coherent sur tout le site
- [ ] Identique a Google My Business
- [ ] Dans le footer

## 5. Liens

### Maillage interne

- [ ] Liens contextuels dans le contenu
- [ ] Menu de navigation coherent
- [ ] Fil d'Ariane
- [ ] Liens footer

### Backlinks

- [ ] Annuaires professionnels
- [ ] Google My Business
- [ ] Partenaires locaux

## 6. Mobile

- [ ] Design responsive
- [ ] Taille de police lisible (min 16px)
- [ ] Boutons tapables (min 44px)
- [ ] Pas de contenu cache
- [ ] Test Mobile-Friendly Google

## 7. Securite

- [ ] HTTPS obligatoire
- [ ] Redirections HTTP > HTTPS
- [ ] Certificat SSL valide

## 8. Vitesse

### Images

- [ ] Format WebP
- [ ] Compression
- [ ] Lazy loading
- [ ] Dimensions specifiees

### Code

- [ ] CSS minifie
- [ ] JS minifie
- [ ] Cache navigateur
- [ ] Compression GZIP

## 9. Suivi

### Outils

- [ ] Google Search Console configure
- [ ] Google Analytics 4 configure
- [ ] RankMath/Yoast configure

### Metriques a surveiller

- [ ] Positions mots-cles principaux
- [ ] Trafic organique
- [ ] CTR SERP
- [ ] Pages indexees
- [ ] Erreurs d'exploration
